# Steven Cheong App Portfolio

A static GitHub Pages portfolio for showcasing selected apps and systems to employers.

Live portfolio: https://tichubrothers.github.io/portfolio/

## Files

- `index.html` contains the page structure and metadata.
- `styles.css` contains the full responsive visual system.
- `scripts.js` contains profile details, project data, filters, search, case studies, and the CV summary copy action.
- `assets/portfolio-hero.png` is the generated hero image used by the site.
- `.github/workflows/pages.yml` deploys the static site through GitHub Pages Actions.

## Edit Before Publishing

Review `scripts.js` and update any project copy, links, or contact details you want to make public.

## Publish To GitHub Pages

1. Create a GitHub repository for this folder.
2. Push the site to the `main` branch.
3. In GitHub, open **Settings > Pages**.
4. Set **Source** to **GitHub Actions**.
5. Push again or run the **Deploy GitHub Pages** workflow manually.

The deployed URL will appear in the workflow summary.
